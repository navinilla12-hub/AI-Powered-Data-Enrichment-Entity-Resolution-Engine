# AI-Powered Data Enrichment & Entity Resolution Engine

## Overview
This project builds a data enrichment and entity resolution system to improve dataset quality by handling missing values, detecting duplicates, and generating enriched features.

## Features
- Data cleaning and preprocessing
- Missing value imputation
- Entity resolution using fuzzy matching
- Feature engineering (Full Name creation)

## Tech Stack
- Python
- Pandas
- Scikit-learn
- FuzzyWuzzy

## Dataset
Public dataset of people records:
https://github.com/datablist/sample-csv-files

## How to Run
1. Open Google Colab
2. Copy and paste the notebook code
3. Run all cells

## Output
- Cleaned dataset
- Duplicate detection results
- Enriched dataset (CSV)

## Use Case
Applicable to:
- Customer data platforms
- Healthcare provider deduplication
- CRM systems
